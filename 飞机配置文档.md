# 飞机配置文档

## 概述

所有飞机（包括玩家和敌人）都使用统一的配置系统，每个飞机对象包含以下属性：

## 飞机属性

| 属性名 | 类型 | 说明 |
|--------|------|------|
| id | String | 飞机唯一标识 |
| name | String | 飞机名称 |
| team | String | 阵营："player" 或 "enemy" |
| speed | float | 移动速度（像素/秒） |
| health | int | 血量 |
| fire_rate | float | 发射子弹间隔（秒） |
| bullet_type | String | 子弹类型ID |
| skill_type | String | 技能类型ID（可选） |
| coin_value | int | 死亡后掉落金币价值 |
| texture | String | 飞机纹理路径 |
| color | Color | 飞机颜色（用于区分） |

## 子弹类型配置

| 子弹类型ID | 速度 | 伤害 | 大小 | 颜色 | 说明 |
|------------|------|------|------|------|------|
| basic | 800 | 1 | 0.2 | 黄色 | 基础子弹 |
| fast | 1200 | 1 | 0.15 | 蓝色 | 快速子弹 |
| heavy | 600 | 3 | 0.3 | 红色 | 重型子弹 |
| spread | 700 | 1 | 0.2 | 绿色 | 散射子弹 |

## 技能类型配置

| 技能类型ID | 冷却时间 | 说明 |
|------------|----------|------|
| none | - | 无技能 |
| dash | 3.0 | 冲刺 |
| shield | 5.0 | 护盾 |
| rapid_fire | 4.0 | 快速射击 |
